# Proyecto VibeCoding

## Prompt 1

Quiero que generes una landing page completa usando solo:
HTML
CSS
JavaScript Vanilla

Temática:
Landing page sobre próximos lanzamientos de videojuegos.

Requisitos:
- Diseño moderno
- Responsive (móvil, tablet y PC)
- Tarjetas con imagen, título, fecha y plataforma
- Barra de búsqueda
- Filtro por plataforma
- Modo oscuro
- Animaciones suaves
- Todo el código separado en:
  index.html
  styles.css
  script.js

Entrégame el contenido completo de cada archivo.

## Prompt 2

Añade un botón para ordenar los videojuegos por fecha y mejora la animación de las tarjetas.

## Prompt 3

Quiero añadir un gráfico interactivo que muestre cuántos videojuegos se lanzan por mes.

El gráfico debe hacerse en JavaScript sin librerías externas.

Debe ser responsive y mantener el diseño actual.
